# 订阅管理

![](https://s3.jpg.cm/2020/07/17/UFfEH.png)

高考几个月之前不幸入坑MJJ，买了不少小鸡，然后因为管理过于麻烦就萌生了写一个方便一起管理的东东。高考完了之后天天在摸鱼，有一天睡前突然想起来有这么个想法。然后就边摸鱼边写，算是摸出来了点东西。



### 食用方法

1. 随便找个PHP+MYSQL环境放上去
2. 将根目录的数据库文件导入数据库
3. 去CONF下修改配置文件
4. 有邮件提示需求的可以设置定时任务定时每天访问一次根目录下的`A.php`



### 功能

- 小鸡信息聚合
- 到期小鸡续费提醒
- 一定程度防止剁手（养鸡成本）
